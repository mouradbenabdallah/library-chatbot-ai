# 📚 Library Chatbot AI

Application de gestion de bibliothèque avec chatbot IA local (Mistral via Ollama).

## Stack

- **Backend** : Flask + SQLAlchemy
- **Base de données** : PostgreSQL
- **Interface** : CustomTkinter
- **IA** : Mistral (Ollama, local)

## Installation

```bash
pip install -r requirements.txt
python run.py
```
